# flower-shop-server
基于SpringBoot+Vue前后端分离的鲜花商城，系统包含管理员、用户角色，功能如下： 管理员：首页统计（商品统计、7天销售金额统计）；用户管理；商品种类；商品管理；订单管理；评论管理；系统留言；系统公告；个人信息；密码修改。 用户：注册、登录系统；首页；鲜花商城（商品分类、商品列表、详情、购物车、收藏、下单、评论）；系统留言；系统公告；用户还具有个人中心功能，功能包括：商品收藏、我的订单、商品评论、系统留言、个人资料、修改密码。 系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，有需要可提供指导。
